# Gestión de Torneo de Fútbol

Proyecto del segundo bimestre – Programación  
Carrera: Tecnología Superior en Desarrollo de Software

## Descripción
Aplicación CRUD desarrollada en C++ con Qt para la gestión de un torneo de fútbol.

## Funcionalidades previstas
- Registro de equipos
- Listado de equipos participantes
- Edición y eliminación de equipos
- Simulación de fases del torneo (grupos, octavos, semifinal y final)

## Herramientas
- C++
- Qt Creator
- GitHub
- Archivos .txt para almacenamiento
